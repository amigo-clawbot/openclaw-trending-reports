# OpenClaw Trending Reports

Public bilingual trend reports generated from public GitHub and Hacker News
metadata.

## Scope

This repository contains sanitized public reports only.

Included:

- public project links;
- public GitHub metadata;
- public Hacker News links and discussion counts;
- English and Chinese summaries.

Excluded:

- private scoring weights;
- owner-specific watchlists or strategy;
- internal OpenClaw infrastructure context;
- private follow-up tasks;
- tokens, keys, logs, or runtime state.

## Boundary

Popularity and discussion volume are weak signals. These reports are a triage
feed, not an endorsement or investment recommendation.
